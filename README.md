# Installation
- Download the quizapp project folder
- cd quizapp/
- npm install
- npm start -> This command will start application in development environment

# Description
Frontend for simple mobile friendly(responsive) quiz 

# Features
- React
- Redux
- React Router v4
- React-bootstrap
